Technical Summary of Changes for Creo 11.0.0.0
This section describes the critical and miscellaneous technical changes in Creo 11.0.0.0 and Creo Object TOOLKIT C++. It also lists the new and superseded functions for this release.
Critical Technical Changes
This section describes the changes in Creo 11.0.0.0 and Creo Object TOOLKIT C++ that might require alteration of existing Creo Object TOOLKIT C++ applications.
Support for Visual Studio
In Creo Parametric 11.0.0.0 and later, Creo Object TOOLKIT C++ supports Visual Studio 2022. The compiler flags and libraries are available for Visual Studio 2022. Creo Object TOOLKIT C++ no longer supports Visual Studio 2019.
All Creo Object TOOLKIT C++ applications on 64-bit Windows platforms built using the Microsoft Visual Studio 2022 compiler must set the configuration property Platform Toolset as Visual Studio 2022 (v17.6.5).
New Functions
This section describes new functions for Creo Object TOOLKIT C++ for Creo 11.0.0.0.
Kinematic Dragging
New Function
Description
wfcWSession::StartKinDrag
Starts a drag session.
wfcWSession::StopKinDrag
Stops the drag session.
wfcWSession::SetKinDragEnvironment
Sets the environment for dragging and reconnect operations.
wfcWSession::MoveKinDrag
Drags the selected geometric object to the specified X and Y screen coordinates.
wfcWSession::MoveKinDragForModel
Brings to drag the selected object.
wfcWSession::CreateSnapshot
Creates a new snapshot for the top-level assembly as per its current position in the Creo Parametric window
wfcWSession::UpdateSnapshot
Updates the snapshot for the top-level assembly as per its current position in the Creo Parametric window.
wfcWSession::DeleteSnapshot
Deletes a specified snapshot.
wfcWSession::GetKinSnapshotTransformations
Retrieves the transformation paths and transformation matrices saved in the snapshots for the subassemblies and their components with respect to the top-level assembly snapshot.
