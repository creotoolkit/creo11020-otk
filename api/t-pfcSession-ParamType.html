Enum pfcParamType

# include <pfcSession.h>

enum pfcParamType
{
    pfcUSER_PARAM,
    pfcDIM_PARAM,
    pfcPATTERN_PARAM,
    pfcDIMTOL_PARAM,
    pfcREFDIM_PARAM,
    pfcALL_PARAMS,
    pfcGTOL_PARAM,
    pfcSURFFIN_PARAM,
    pfcParamType_nil
};

Description

Enumeration of parameters types that is used to specify which parameters to display.
See Also:
pfcSession::UIDisplayFeatureParams(pfcSelection_ptr, pfcParamType)



Values

pfcUSER_PARAM
Displays user parameters.
pfcDIM_PARAM
Displays dimension parameters.
pfcPATTERN_PARAM
Displays pattern parameters.
pfcDIMTOL_PARAM
Displays dimension tolerance parameters.
pfcREFDIM_PARAM
Displays reference dimension parameters.
pfcALL_PARAMS
Displays all parameters.
pfcGTOL_PARAM
Displays geometric tolerance parameters.
pfcSURFFIN_PARAM
Displays surface finish parameters.
pfcParamType_nil
Enum null value.
