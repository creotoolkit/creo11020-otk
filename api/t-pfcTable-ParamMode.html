Enum pfcParamMode

# include <pfcTable.h>

enum pfcParamMode
{
    pfcDWGTABLE_NORMAL,
    pfcDWGTABLE_FULL,
    pfcParamMode_nil
};

Description

Describes the options used when obtaining the text from a table cell.
See Also:
pfcTable::GetText(pfcTableCell_ptr, pfcParamMode)



Values

pfcDWGTABLE_NORMAL
Retrieve the text in normal mode (as it is shown on the screen).
pfcDWGTABLE_FULL
Retrieve the text in full symbolic mode.
pfcParamMode_nil
Enum null value.
