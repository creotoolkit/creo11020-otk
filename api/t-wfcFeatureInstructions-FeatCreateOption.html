Enum wfcFeatCreateOption

# include <wfcFeatureInstructions.h>

enum wfcFeatCreateOption
{
    wfcFEAT_CR_NO_OPTS,
    wfcFEAT_CR_DEFINE_MISS_ELEMS,
    wfcFEAT_CR_INCOMPLETE_FEAT,
    wfcFEAT_CR_FIX_MODEL_ON_FAIL,
    wfcFEAT_CR_DO_NOT_DISPLAY,
    wfcFEAT_CR_CALLED_FROM_TK,
    wfcFEAT_CR_UNDO_IF_FAIL,
    wfcFEAT_CR_ALLOW_CONFIRM,
    wfcFeatCreateOption_nil
};

Description

Various options used while feature creation and redefination
Manual References:
Feature Element Tree: Creating FET Using WCreateFeature



Values

wfcFEAT_CR_NO_OPTS
No options
wfcFEAT_CR_DEFINE_MISS_ELEMS
Prompt user interface upon feature creation or redefinition if some of the required elements are missing.
wfcFEAT_CR_INCOMPLETE_FEAT
Allow incomplete feature creation or redefination
wfcFEAT_CR_FIX_MODEL_ON_FAIL
If there is a feature failure then prompt fix model interface
wfcFEAT_CR_DO_NOT_DISPLAY
Do not redisplay model and model tree after feature creation or redefinition.
wfcFEAT_CR_CALLED_FROM_TK
Internal use only.
wfcFEAT_CR_UNDO_IF_FAIL
Internal use only.
wfcFEAT_CR_ALLOW_CONFIRM
Internal use only.
wfcFeatCreateOption_nil
Enum null value.
