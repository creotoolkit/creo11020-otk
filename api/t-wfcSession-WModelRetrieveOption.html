Enum wfcWModelRetrieveOption

# include <wfcSession.h>

enum wfcWModelRetrieveOption
{
    wfcMODEL_RETRIEVE_NONE,
    wfcMODEL_RETRIEVE_NORMAL,
    wfcMODEL_RETRIEVE_SIMP_REP,
    wfcMODEL_RETRIEVE_VIEW_ONLY,
    wfcWModelRetrieveOption_nil
};

Description

Provides options to retrieve model.
Manual References:
Action Listeners: Model Level Action listeners



Values

wfcMODEL_RETRIEVE_NONE
Do not retrieve model
wfcMODEL_RETRIEVE_NORMAL
Retrieve normally
wfcMODEL_RETRIEVE_SIMP_REP
Retrieve in simplified representation.
wfcMODEL_RETRIEVE_VIEW_ONLY
Retrieve in view only mode.Valid for drawings only
wfcWModelRetrieveOption_nil
Enum null value.
