Enum wfcFamilyTableStatus

# include <wfcFamily.h>

enum wfcFamilyTableStatus
{
    wfcFAMILY_TABLE_NONE,
    wfcFAMILY_TABLE_VALID_AND_NOT_EMPTY,
    wfcFAMILY_TABLE_VALID_AND_EMPTY,
    wfcFamilyTableStatus_nil
};

Description

Check family table validity.



Values

wfcFAMILY_TABLE_NONE
No family table is found.
wfcFAMILY_TABLE_VALID_AND_NOT_EMPTY
Family table is found and is not empty.
wfcFAMILY_TABLE_VALID_AND_EMPTY
Family table is found but is empty.
wfcFamilyTableStatus_nil
Enum null value.
