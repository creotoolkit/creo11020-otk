Enum pfcDisplayStatus

# include <pfcLayer.h>

enum pfcDisplayStatus
{
    pfcLAYER_NORMAL,
    pfcLAYER_DISPLAY,
    pfcLAYER_BLANK,
    pfcLAYER_HIDDEN,
    pfcDisplayStatus_nil
};

Description

This enumerated type lists the possible values for the display status of a layer.
Manual References:
ModelItem: Layer Operations



Values

pfcLAYER_NORMAL
A normal layer.
pfcLAYER_DISPLAY
A layer selected for display.
pfcLAYER_BLANK
A blanked layer.
pfcLAYER_HIDDEN
A hidden layer. This applies to Assembly mode only.
pfcDisplayStatus_nil
Enum null value.
