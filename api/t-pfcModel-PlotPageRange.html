Enum pfcPlotPageRange

# include <pfcModel.h>

enum pfcPlotPageRange
{
    pfcPLOT_RANGE_ALL,
    pfcPLOT_RANGE_CURRENT,
    pfcPLOT_RANGE_OF_PAGES,
    pfcPlotPageRange_nil
};

Description

This enumerated type specifies which pages to plot.



Values

pfcPLOT_RANGE_ALL
Plot all the pages.
pfcPLOT_RANGE_CURRENT
Plot the current page.
pfcPLOT_RANGE_OF_PAGES
Plot a range of pages.
pfcPlotPageRange_nil
Enum null value.
