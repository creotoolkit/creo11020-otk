Function pfcCreateASCIIStringArgValue

# include <pfcArgument.h>

pfcArgValue_ptr pfcCreateASCIIStringArgValue (xrstring Value)


Description

Creates an ASCII string argument value object.

This type will be passed to Creo Parametric TOOLKIT using the ProValueDataType of PRO_VALUE_TYPE_STRING.
See Also:
pfcArgument::Create(xrstring, pfcArgValue_ptr), pfcDll::ExecuteFunction(xrstring, pfcArguments_ptr)
Parameters:
Value
The character string.
Returns:
The created argument object.
