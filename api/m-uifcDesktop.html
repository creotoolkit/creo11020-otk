File "uifcDesktop.h"

Description

The Desktop is a special class which represents the desktop of the machine upon which the application is being displayed.
The Desktop supports a simple drawing API to allow lines and shapes to be drawn XORed with the system desktop.
The Desktop has no associated resource file.
