If true the children of feature, if members of a group, will be individually suppressed from their group. If false, then the group containing the feature's children will be also suppressed. This attribute can be set to true if and only if both pfcSuppressOperation::GetClip(xbool) and pfcSuppressOperation::GetAllowGroupMembers(xbool) are set to true. Default value is false.
