If true the children of feature, if members of a group, will be individually deleted from their group. If false, then the group containing the feature's children will be also deleted. This attribute can be set to true if and only if both pfcDeleteOperation::GetClip(xbool) and pfcDeleteOperation::GetAllowGroupMembers(xbool) are set to true. Default value is false.
