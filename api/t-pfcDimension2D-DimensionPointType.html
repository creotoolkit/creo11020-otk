Enum pfcDimensionPointType

# include <pfcDimension2D.h>

enum pfcDimensionPointType
{
    pfcDIMPOINT_END1,
    pfcDIMPOINT_END2,
    pfcDIMPOINT_CENTER,
    pfcDIMPOINT_NONE,
    pfcDIMPOINT_MIDPOINT,
    pfcDimensionPointType_nil
};

Description

This enumerated type contains the types of point references used by a pfcPointDimensionSense object.
See Also:
pfcPointDimensionSense, pfcDimension2D::GetDimensionSenses(), pfcPointDimensionSense::Create(pfcDimensionPointType)



Values

pfcDIMPOINT_END1
The entity endpoint where t=0.0
pfcDIMPOINT_END2
The entity endpoint where t=1.0
pfcDIMPOINT_CENTER
The entity center point (arc or circle entities).
pfcDIMPOINT_NONE
No infomation was provided about the attachment point for this entity.
pfcDIMPOINT_MIDPOINT
The entity midpoint where t=0.5
pfcDimensionPointType_nil
Enum null value.
