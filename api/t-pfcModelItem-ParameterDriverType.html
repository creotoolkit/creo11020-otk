Enum pfcParameterDriverType

# include <pfcModelItem.h>

enum pfcParameterDriverType
{
    pfcPARAMDRIVER_PARAM,
    pfcPARAMDRIVER_FUNCTION,
    pfcPARAMDRIVER_RELATION,
    pfcParameterDriverType_nil
};

Description

This enumerated type lists the possible parameter driver type.
Manual References:
Dimensions and Parameters: Driven and Driving Parameters



Values

pfcPARAMDRIVER_PARAM
The parameter value is driven by another parameter.
pfcPARAMDRIVER_FUNCTION
The parameter value is driven by a function.
pfcPARAMDRIVER_RELATION
The parameter value is driven by a relation.
pfcParameterDriverType_nil
Enum null value.
