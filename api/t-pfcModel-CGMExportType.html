Enum pfcCGMExportType

# include <pfcModel.h>

enum pfcCGMExportType
{
    pfcEXPORT_CGM_CLEAR_TEXT,
    pfcEXPORT_CGM_MIL_SPEC,
    pfcCGMExportType_nil
};

Description

Enumeration that indicates whether a CGM export file should be ASCII (clear text) or binary (mil spec).

For more information about exporting CGM files, refer to the Interface section of the Creo Parametric help.
See Also:
pfcCGMFILEExportInstructions::Create(pfcCGMExportType, pfcCGMScaleType), pfcModel::Export(xrstring, pfcExportInstructions_ptr)



Values

pfcEXPORT_CGM_CLEAR_TEXT
Passed to pfcCGMFILEExportInstructions::Create(pfcCGMExportType, pfcCGMScaleType) to indicate that a CGM export file should be ASCII.
pfcEXPORT_CGM_MIL_SPEC
Passed to pfcCGMFILEExportInstructions::Create(pfcCGMExportType, pfcCGMScaleType) to indicate that a CGM export file should be binary.
pfcCGMExportType_nil
Enum null value.
