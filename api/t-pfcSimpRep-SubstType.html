Enum pfcSubstType

# include <pfcSimpRep.h>

enum pfcSubstType
{
    pfcSUBST_INTERCHG,
    pfcSUBST_PRT_REP,
    pfcSUBST_ASM_REP,
    pfcSUBST_ENVELOPE,
    pfcSUBST_NONE,
    pfcSubstType_nil
};

Description

This enumerated type lists the supported types of substitution.



Values

pfcSUBST_INTERCHG
Substitute using an interchange assembly or a family table.
pfcSUBST_PRT_REP
Substitute a part with a simplified representation.
pfcSUBST_ASM_REP
Substitute an assembly with a simplified representation.
pfcSUBST_ENVELOPE
Substitute with an envelope.
pfcSUBST_NONE
 
pfcSubstType_nil
Enum null value.
