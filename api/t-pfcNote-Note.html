DEPRECATED: since Creo 4

The text of the note.

Using this to read the text of the note will always get the text in numeric form (parameter callouts will not be identified). To get the text with the parameter callouts, use pfcNote::GetText(xbool)

Exceptions thrown (but not limited to):

pfcXToolkitBadContext - The note text cannot be varied.
