Enum pfcShrinkwrapFacetedFormat

# include <pfcShrinkwrap.h>

enum pfcShrinkwrapFacetedFormat
{
    pfcSWFACETED_PART,
    pfcSWFACETED_STL,
    pfcSWFACETED_VRML,
    pfcSWFACETED_LIGHTWEIGHT_PART,
    pfcShrinkwrapFacetedFormat_nil
};

Description

This enumerated type contains the possible formats for the export of a solid model to a faceted shrinkwrap format.
See Also:
pfcSolid::ExportShrinkwrap(pfcShrinkwrapExportInstructions_ptr), pfcShrinkwrapFacetedPartInstructions, pfcShrinkwrapSTLInstructions, pfcShrinkwrapVRMLInstructions



Values

pfcSWFACETED_PART
Export the model to a Creo Parametric part (or lightweight part).
pfcSWFACETED_STL
Export the model to STL.
pfcSWFACETED_VRML
Export the model to VRML.
pfcSWFACETED_LIGHTWEIGHT_PART
 
pfcShrinkwrapFacetedFormat_nil
Enum null value.
