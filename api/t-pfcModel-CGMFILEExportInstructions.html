This is a COMPACT class.

Description

Used to export a drawing in CGM format.

Using this export class is equivalent to using the File > Save As > Export > CGM option in Creo Parametric. For more information, refer to the Interface section of the Creo Parametric on-line help.

Creates a new instructions object used to export a drawing in CGM format.

Using this export class is equivalent to using the File > Save As > Export > CGM option in Creo Parametric. For more information, refer to the Interface section of the Creo Parametric on-line help. This argument must be passed to pfcModel::Export(xrstring, pfcExportInstructions_ptr) to produce an output file. Valid only for solid models.
