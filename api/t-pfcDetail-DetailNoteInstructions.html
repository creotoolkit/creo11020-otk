# include <pfcDetail.h>

 : public virtual pfcDetailCreateInstructions

    xdeclare(pfcDetailNoteInstructions)
    ...
};
This is a COMPACT class.

Description

This interface contains information used to describe a detail note item. It may be used when creating detail notes (pfcDetailItemOwner::CreateDetailItem(pfcDetailCreateInstructions_ptr) and pfcDetailSymbolDefItem::CreateDetailItem(pfcDetailCreateInstructions_ptr)), or accessing existing notes (pfcDetailNoteItem::Modify(pfcDetailNoteInstructions_ptr) and pfcDetailNoteItem::GetInstructions(xbool)).
