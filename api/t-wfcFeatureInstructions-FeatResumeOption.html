Enum wfcFeatResumeOption

# include <wfcFeatureInstructions.h>

enum wfcFeatResumeOption
{
    wfcFEAT_RESUME_NO_OPTS,
    wfcFEAT_RESUME_INCLUDE_PARENTS,
    wfcFEAT_RESUME_ADD_TO_BUFFER,
    wfcFeatResumeOption_nil
};

Description

Various options used while resuming feature



Values

wfcFEAT_RESUME_NO_OPTS
No options
wfcFEAT_RESUME_INCLUDE_PARENTS
Suppressed parents of the feature will also be resumed
wfcFEAT_RESUME_ADD_TO_BUFFER
Add the resumed features to the selection buffer.
wfcFeatResumeOption_nil
Enum null value.
