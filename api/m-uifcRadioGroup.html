File "uifcRadioGroup.h"

Description

A RadioGroup is a selection component which contain buttons defined with the items of ItemNameArray, ItemTextArray and ItemImageArray.
The SelectedItemNameArray contains 1 item which controls which button is selected.
Each button may contain both an image and a label which may be displayed in a number of ways relative to each other.
