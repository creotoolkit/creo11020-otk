Enum pfcDimensionType

# include <pfcDimension.h>

enum pfcDimensionType
{
    pfcDIM_LINEAR,
    pfcDIM_RADIAL,
    pfcDIM_DIAMETER,
    pfcDIM_ANGULAR,
    pfcDimensionType_nil
};

Description

This enumerated type lists the possible dimension types.



Values

pfcDIM_LINEAR
A linear dimension.
pfcDIM_RADIAL
A radial dimension.
pfcDIM_DIAMETER
A diametrical dimension.
pfcDIM_ANGULAR
An angular dimension.
pfcDimensionType_nil
Enum null value.
