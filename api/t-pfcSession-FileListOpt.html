Enum pfcFileListOpt

# include <pfcSession.h>

enum pfcFileListOpt
{
    pfcFILE_LIST_ALL,
    pfcFILE_LIST_LATEST,
    pfcFILE_LIST_ALL_INST,
    pfcFILE_LIST_LATEST_INST,
    pfcFileListOpt_nil
};

Description

This enumerated type indicates which files should be returned when listing files in a directory.
See Also:
pfcBaseSession::ListFiles(xrstring, pfcFileListOpt, optional xrstring)
Manual References:
Session Objects: File Handling



Values

pfcFILE_LIST_ALL
List all files (including multiple versions of the same file).
pfcFILE_LIST_LATEST
List only the latest version of each file.
pfcFILE_LIST_ALL_INST
Same as FILE_LIST_ALL, to show instances for PDM locations.
pfcFILE_LIST_LATEST_INST
Same as FILE_LIST_LATEST, to show instances for PDM locations.
pfcFileListOpt_nil
Enum null value.
