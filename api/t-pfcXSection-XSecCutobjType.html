Enum pfcXSecCutobjType

# include <pfcXSection.h>

enum pfcXSecCutobjType
{
    pfcXSECTYPE_MODEL,
    pfcXSECTYPE_QUILTS,
    pfcXSECTYPE_MODELQUILTS,
    pfcXSECTYPE_ONEPART,
    pfcXSecCutobjType_nil
};

Description

This enumerated type describes the type of objects intersected by a cross section.
See Also:
pfcXSection::GetXSecType()
Manual References:
Cross Sections: Listing Cross Sections, Cross Sections: Creating and Modifying Cross Sections



Values

pfcXSECTYPE_MODEL
The cross section intersects the solid model.
pfcXSECTYPE_QUILTS
The cross section intersects all quilts.
pfcXSECTYPE_MODELQUILTS
The cross section intersects the model and all quilts.
pfcXSECTYPE_ONEPART
 
pfcXSecCutobjType_nil
Enum null value.
